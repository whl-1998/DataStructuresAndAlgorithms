### 1. 贪心算法
思路：
```java
public int canCompleteCircuit2(int[] gas, int[] cost) {
    int total_tank = 0;
    int curr_tank = 0;
    int startStation = 0;

    for (int i = 0; i < gas.length; i++) {
        total_tank += gas[i] - cost[i];
        curr_tank += gas[i] - cost[i];
        if (curr_tank < 0) {
            startStation = i + 1;
            curr_tank = 0;
        }
    }
    return total_tank >= 0 ? startStation : -1;
}
```
